Alumni Management System
📖 Overview

The Alumni Management System is a web-based application designed to manage and maintain alumni information efficiently. It provides a platform where alumni can register, update their profiles, connect with fellow graduates, view events, and stay connected with their educational institution.

The system helps educational institutions build a strong alumni network and improve communication between alumni and current students.

✨ Features
User Registration and Login
Alumni Profile Management
Alumni Directory Search
Event Management
News and Announcements
Job Opportunities Posting
Admin Dashboard
Secure Authentication
Responsive User Interface
